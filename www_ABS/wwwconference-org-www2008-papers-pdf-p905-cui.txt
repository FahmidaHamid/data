this paper describes a series of user  studies  on  how  people  use the  web  via  mobile  devices.  the  data  primarily  comes  from contextual inquiries with 47 participants between 2004 and 2007, and is complemented with a phone log analysis of 577 panelists in 2007. we report four key contextual factors in using the web on mobile devices and propose mobile web activity taxonomy. the framework  contains  three  user  activity  categories  identical  to seeking, previous communication,  and  transaction,  and  a  new  category  personal space  extension.  the  new  category  refers  to  the  practice  that people put their content on the web for personal access, therefore extending their personal information space. stationary  web information studies
